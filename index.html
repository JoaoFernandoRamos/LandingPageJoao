Matérias na Faculdade

Atualmente, estou cursando as seguintes matérias na faculdade:

  • Experiência e Interface com o Usuário: Esta matéria aborda princípios de design de interfaces de usuário, usabilidade e experiência do usuário (UX). Aprenderemos a criar interfaces amigáveis e eficazes.
  • Design e Desenvolvimento de Banco de Dados I: Nesta matéria, exploramos os conceitos fundamentais de design de banco de dados e SQL. Aprenderemos a criar e gerenciar bancos de dados relacionais.
  • Construção de Software para Web: Aqui, focamos no desenvolvimento de aplicativos web. Exploraremos tecnologias front-end (HTML, CSS, JavaScript) e back-end (Node.js, Express) para criar aplicações web interativas.
  • Estrutura de Computadores: Esta matéria trata da arquitetura de computadores e dos componentes de hardware. Aprenderemos como os computadores funcionam em nível de hardware.
  • Gestão de Pessoas: Nesta disciplina, exploramos princípios de gestão de recursos humanos. Aprenderemos sobre liderança, motivação e gerenciamento de equipes.
  • Fundamentos de Tecnologia da Computação: Esta matéria abrange os conceitos básicos da ciência da computação, incluindo algoritmos, estruturas de dados e programação. É uma base fundamental para nossa formação em Ciências da Computação.
